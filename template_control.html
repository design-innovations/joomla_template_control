Question: Can you make my site fit into my mobile screen?

Answer: Simple meta tag

Add this code into the head of your template.

$doc = JFactory::getDocument();
$doc->setMetaData( 'HandheldFriendly', 'True' );
$doc->setMetaData( 'MobileOptimized', '320' );
$doc->setMetaData( 'viewport', 'width=device-width, initial-scale=1.0' );

Question: Can you insert my Facebook tracking code into my site?

Answer: Add this code below jdoc:include type=”head”

$doc = JFactory::getDocument();
$doc->addCustomTag( '
    <meta property="fb:page_id" content="' . JRequest::getInt('id') . '" />
    <meta property="og:type" content="article" />
    <meta property="og:url" content="' . JURI::current() . '" />
    <meta property="og:image" content="http://www.yoursite.com/templates/yourtemplate/images/inside-logo.gif" />
    <meta property="og:site_name" content="Your Site Name" />
' );

Question: Can I have my frontpage styled differently than the rest of my site?

Answer: Add the following PHP into the head and replace body with last line

<?php
$menu = JFactory::getApplication()->getMenu();
$body = 'subpage';
if ( $menu->getActive() == $menu->getDefault() ) {
    $body = 'frontpage';
}
?>

<body class=”<?php echo $body; ?>”>

Now you can add the following into your CSS to have a different color header for frontpage

.frontpage #header {background: #BC4B4A}

Question: How do I stop MooTools from loading in my template?

Answer: Use the following code to remove all instances of MooTools from loading

$doc = JFactory::getDocument();
unset($doc->_scripts[$this->baseurl.'/media/system/js/mootools-core.js']);
    unset($doc->_scripts[$this->baseurl.'/media/system/js/mootools-more.js']);
    unset($doc->_scripts[$this->baseurl.'/media/system/js/core.js']);
    unset($doc->_scripts[$this->baseurl.'/media/system/js/caption.js']);
    unset($doc->_scripts[$this->baseurl.'/media/system/js/modal.js']);
    unset($doc->_scripts[$this->baseurl.'/media/system/js/mootools.js']);
    unset($doc->_scripts[$this->baseurl.'/plugins/system/mtupgrade/mootools.js']);

Question: How can I keep all standard Joomla! javascript from loading in my template?

Answer: Use the following code to remove all javascript from loading

$doc = JFactory::getDocument();
  unset($doc->_scripts[$this->baseurl.'/media/jui/js/jquery.min.js']);
  unset($doc->_scripts[$this->baseurl.'/media/jui/js/jquery-noconflict.js']);
  unset($doc->_scripts[$this->baseurl.'/media/jui/js/jquery-migrate.min.js']);
  unset($doc->_scripts[$this->baseurl.'/media/jui/js/bootstrap.min.js']);
  unset($doc->_scripts[$this->baseurl.'/media/system/js/tabs-state.js']);

This is handy if you are replacing default scripts with your own stripped down versions and want fine-grain control of what is loading.

Question: What are ways I can debug my code easily?

Answer: There are many ways to debug, but a very quick way is to dump the last error that happens to your screen:

Insert the following code at the top of your index.php file:

register_shutdown_function(
    function () {
        $error = error_get_last();
        var_dump($error);
    }
);

Question: How can I modify the output of module containers?

Answer: Modify the modules.php file in your template HTML folder

There are times when the standard module output doesn’t work for you. Luckily, Joomla! has done a fantastic job of giving developers full control of what outputs. All files that you need to override need to be added to the html folder within your template folder. The file you will want to modify is located at templates > your_module_name > html > modules.php. Joomla! will automatically load the file at templates > system > html > modules.php so you can look in that file to see all of the available styles you can use. Look at the name of the function for the style name. In the following example the name of the name of the style you would use is style=”html5”. 

function modChrome_html5($module, &$params, &$attribs)

If, for example, you wanted to modify the output of the standard xhtml style then you would copy the entire module code into the module.php file in your template’s html folder.

function modChrome_xhtml($module, &$params, &$attribs)
{
    if (!empty ($module->content)) : ?>
        <div class="moduletable<?php echo htmlspecialchars($params->get('moduleclass_sfx')); ?>">
        <?php if ((bool) $module->showtitle) : ?>
            <h3><?php echo $module->title; ?></h3>
        <?php endif; ?>
            <?php echo $module->content; ?>
        </div>
    <?php endif;
}

The first step is to modify the function name since you can’t have two function names be the same, so we will change it to myxhtml. Since it’s generally not a good idea to style using a specific H tag, we will add a general class name called “title” to the H3 and then if you ever changed H tags your styles won’t change. To do this you would modify the code in the following way:

function modChrome_myxhtml($module, &$params, &$attribs)
{
    if (!empty ($module->content)) : ?>
        <div class="moduletable<?php echo htmlspecialchars($params->get('moduleclass_sfx')); ?>">
        <?php if ((bool) $module->showtitle) : ?>
            <h3 class="title"><?php echo $module->title; ?></h3>
        <?php endif; ?>
            <?php echo $module->content; ?>
        </div>
    <?php endif;
}

If you really want to make your modules shine, then you can change them to use all of the settings that are available within the Advanced tab in the module admin such as H tags and bootstrap grid size.

function modChrome_myhtml($module, &$params, &$attribs)
{ 
    $moduleTag      = htmlspecialchars($params->get('module_tag', 'div'));
    $headerTag      = htmlspecialchars($params->get('header_tag', 'h3'));
    $headerClass    = $params->get('header_class');
    $bootstrapSize  = $params->get('bootstrap_size');
    $moduleClass    = !empty($bootstrapSize) ? ' span' . (int) $bootstrapSize . '' : '';
    $moduleClassSfx = htmlspecialchars($params->get('moduleclass_sfx'));

    if (!empty ($module->content)) {
        $html = "<{$moduleTag} class=\"module{$moduleClassSfx} {$moduleClass}\" id=\"Mod{$module->id}\">" . "<div class=\"module-inner\">";

        if ($module->showtitle != 0) {
            $html .= "<{$headerTag} class=\"module-title {$headerClass}\"><span>{$module->title}</span></{$headerTag}>";
        }

        $html .= "<div class=\"module-content\">{$module->content}</div></div></{$moduleTag}>";

        echo $html;
    }
}

Now we have a module with some power! Any setting you do in the Advanced tab will change the module accordingly as well as having a few extra hooks added to style almost any way you want.

Question: How can I make my columns dynamic according to how many modules I have in the row?

Answer: The following PHP will dynamically change the column widths according to which columns are filled.

<?php 
      // Set defaults
      $column1 = 4;
      $column2 = 4;
      $column3 = 4;

      if ($this->countModules('column1') OR $this->countModules('column2') OR $this->countModules('column3')) {
        $column1 = 12;
        $column2 = 12;
        $column3 = 12;
      }
      if (
        ($this->countModules('column1') AND $this->countModules('column2')) OR
        ($this->countModules('column1') AND $this->countModules('column3')) OR
        ($this->countModules('column2') AND $this->countModules('column3')) ) {
        $column1 = 4;
        $column2 = 4;
        $column3 = 4;
      }
      if (
        ($this->countModules('column1') AND $this->countModules('column2') AND $this->countModules('column3')) ) {
        $column1 = 4;
        $column2 = 4;
        $column3 = 4;
      }

      // Pull in parameters from template for custom settings
      $variable_width = $this->params->get('column_width');
      if ( $variable_width == '6_3_3') {
        $column1 = 6;
        $column2 = 3;
        $column3 = 3;
      }
      if ( $variable_width == '8_2_2') {
        $column1 = 8;
        $column2 = 2;
        $column3 = 2;
      }
    ?>

    <div class="row">
      <?php if ($this->countModules('column1')) : ?>
      <div class="span<?php echo $column1; ?> pull-right">
        <jdoc:include type="modules" name="column1" style="xhtml" />
      </div>
      <?php endif; ?>

      <?php if ($this->countModules('column2')) : ?>
      <div class="span<?php echo $column2; ?> push-left">
        <jdoc:include type="modules" name="column2" style="xhtml" />
      </div>
      <?php endif; ?>

      <?php if ($this->countModules('column3')) : ?>
      <div class="span<?php echo $column3; ?>">
        <jdoc:include type="modules" name="column3" style="xhtml" />
      </div>
      <?php endif; ?>
    </div>
